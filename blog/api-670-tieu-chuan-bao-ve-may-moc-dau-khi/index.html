API 670 — Tiêu chuẩn giám sát bảo vệ máy móc trong ngành dầu khí: Yêu cầu cảm biến và hệ thống đo lường

Fast Group Engineering · Cập nhật: Tháng 7, 2026 · 9 phút đọc
Nhập khẩu chính hãng từ Anh Có C/O & C/Q Xuất VAT đầy đủ Kho TPHCM & Vũng Tàu Hỗ trợ kỹ thuật

Trong các dự án dầu khí tại Việt Nam — từ Block B, Cá Voi Xanh đến các nhà máy xử lý khí của PV GAS — mọi thiết bị quay quan trọng đều phải tuân thủ tiêu chuẩn bảo vệ máy móc. API 670 (Machinery Protection Systems, 5th Edition) là tiêu chuẩn ngành dầu khí quốc tế định nghĩa chính xác phải đo gì, đo bằng cách nào và bảo vệ như thế nào cho các máy compressor, turbine và bơm quan trọng.

Phạm vi áp dụng API 670

API 670 áp dụng cho critical rotating machinery trong ngành dầu khí, hóa chất và lọc dầu. Các loại máy điển hình:

Bốn loại đo lường theo API 670

API 670 yêu cầu 4 loại đo lường chính, mỗi loại dùng loại cảm biến khác nhau:

Measurement TypeThông sốLoại cảm biếnGhi chú
Radial vibrationShaft relative vibration (µm pk-pk)Eddy current proximity probe (không tiếp xúc)2 probes × 90° tại mỗi ổ bi
Axial positionThrust bearing position (mm)Eddy current proximity probeThường 1–2 probes, trip khi vượt giới hạn
Seismic vibrationBearing housing absolute vibration (mm/s hay g)Accelerometer / velocity sensorBổ sung hoặc thay thế shaft vibration cho máy nhỏ
Speed / PhaseRPM + phase referenceEddy current hoặc magnetic pickupCần cho phân tích FFT và balancing

Seismic Vibration — Vai trò của Accelerometer Hansford

Trong bốn loại đo lường của API 670, seismic vibration (đo tại vỏ máy, không phải trên trục) thường dùng cảm biến accelerometer IEPE hoặc velocity sensor. Đây là nơi Hansford Sensors đóng vai trò trực tiếp:

  • API 670 khoản 6.3.6: Seismic vibration detectors phải có sensitivity tối thiểu 100 mV/g (hoặc tương đương)
  • API 670 khoản 6.3.7: Thiết bị trong khu vực hazardous phải có chứng nhận phù hợp (ATEX/IECEx)
  • HS-421I (4-20mA, ATEX Zone 1) và HS-100IS (IEPE, ATEX Zone 1) đáp ứng cả hai yêu cầu trên

Alarm và Trip theo API 670

API 670 yêu cầu hai mức ngưỡng cho mỗi điểm đo:

  • Alert (Alarm): Cảnh báo — hệ thống vẫn chạy nhưng cần hành động
  • Danger (Trip): Dừng máy tự động — bảo vệ thiết bị tránh hư hỏng nghiêm trọng

Ngưỡng trip thường đặt tại 125–150% ngưỡng alarm, tùy loại máy và nhà sản xuất. Cụ thể theo từng loại máy tra trong tiêu chuẩn API liên quan (API 670 thường dẫn chiếu API 612, 613, 616, 617...).

Câu hỏi thường gặp

API 670 có bắt buộc dùng proximity probe không?

Với máy nén ly tâm và turbine lớn, API 670 yêu cầu shaft relative vibration = proximity probe. Với máy nhỏ hơn hoặc khi geometry không cho phép đặt probe, API 670 cho phép dùng seismic detector (accelerometer) thay thế, với điều kiện được approved bởi purchaser và manufacturer.

Hansford Sensors có proximity probe không?

Hansford Sensors chuyên về accelerometer và velocity sensor (seismic), không sản xuất eddy current proximity probe. Với dự án cần đầy đủ proximity probe system (shaft relative vibration), Fast Group Engineering tư vấn giải pháp tổng thể kết hợp Hansford (seismic) với probe system từ nhà sản xuất phù hợp.

API 670 5th Edition so với 4th Edition khác gì?

5th Edition (2014) bổ sung thêm: wireless measurement options, updated safety requirements, expanded machinery types, và thêm guidance cho speed/phase detection. Về cơ bản vẫn yêu cầu 4 measurement types giống nhau.

Nhà máy tại Việt Nam có bắt buộc áp dụng API 670 không?

API 670 không phải luật pháp Việt Nam. Tuy nhiên hầu hết EPC Contractor quốc tế (Technip, JGC, Samsung E&C...) và chủ đầu tư dầu khí (PVN, PTSC) áp dụng API 670 như tiêu chuẩn mặc định trong tài liệu thiết kế và mua sắm.
